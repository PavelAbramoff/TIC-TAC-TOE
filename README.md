**TIC-TAC-TOE – это классическая игра для двух игроков, в которой один игрок играет за крестики, а другой за нолики. Игроки поочередно делают ходы на игровом поле 3x3, стремясь разместить три своих символа в одну линию по вертикали, горизонтали или диагонали. Основные особенности приложения включают:**

	•	Режим против компьютера (AI): Играйте против искусственного интеллекта, который использует случайную стратегию для выбора ходов.
	•	Легкий и интуитивный интерфейс: Красивый и простой дизайн делает игру доступной для игроков любого возраста.
	•	Поддержка локальной игры для двух человек: Играйте с друзьями, делая ходы на одном устройстве.
	•	Умный индикатор очередности: Показывает, чей сейчас ход, помогая ориентироваться в игре.
	•	Автоматическая проверка победителя: Игра автоматически определяет, кто выиграл, и объявляет победителя.
	•	Режим ничьей: Игра завершится вничью, если все клетки заполнены, а победитель не определен, либо таймер вышел.

Приложение предлагает отличный способ развлечься и потренировать логику, играя в любимую с детства игру.**
